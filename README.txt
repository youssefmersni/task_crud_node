#  Task Manager CRUD App

A simple full-stack Task Manager app built with Node.js, Express, MongoDB, and EJS. Supports task creation, editing, deletion, and completion status.

##  Features

- Create, read, update, and delete tasks (CRUD)
- Mark tasks as done or undone
- Flash messages for user feedback
- Clean and responsive UI using basic CSS

##  Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS (Embedded JavaScript templates)
- connect-flash & express-session
- dotenv for environment config
